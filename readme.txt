Игра Zuma

Выполнил: Тебайкин Максим

Описание:
Реализована классическая игра Zuma.
Цель игры - уничтожить как можно больше шаров. Для уничтожения шаров нужно выстреливать по ним, собирая их в ряд из не менее 3 штук.

Управление:
Управление производится мышью.
ЛКМ - выстрелить шаром.

Запуск:
python zuma.py
